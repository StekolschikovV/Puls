Проблемы:
1. Длинная изначальная загрузка - происходит получение данных с сервера.
2. Фризы при переходе на вкладку
3. Получение избыточных данных - в получаемых данных множество неиспользуемых данных
---------------------------------------------
Как решить проблему 1?
Без изменения и доработки API ника. При условии доработки API можно реализовать ленивую загрузку данных при прокрутке.
---------------------------------------------
Как решить проблему 2?
В минимальном варианте оптимизация была выполнена. Данные не все выводятся в шаблон, а только те которые сейчас видны. (демонстрация в demonstration.mov)
В оптимальном варианте с изменением API можно добавить ленивую загрузку и перевести страницу на использовании shadow dom.
---------------------------------------------
Как решить проблему 3?
Без изменения доработки API ника. При условии доработки API можно удалить часть данных из ответа или получать их по требования
---------------------------------------------
